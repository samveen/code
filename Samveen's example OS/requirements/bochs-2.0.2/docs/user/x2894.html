Windows XP

You must read the message regarding software licenses in the Section called Third Party Software Licensing and Temporary Files in the chapter called Introduction to Bochs before you install Windows XP as a guest OS in Bochs.

Windows XP has been reported to install from the cd, and run inside Bochs. The only known issue is to set the IPS to, at least, a value of 10000000.
